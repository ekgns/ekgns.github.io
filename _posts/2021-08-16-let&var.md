### 상수 : 고정 된 수
-   사용방법 : 
```
1. let (이름) : <Type> : 값
2. let (이름) = 값
```
-   ex)
``` 
1.  let name : String =  Bean
(상수) (상수이름) : <타입> =  값
2.  let name2  = Bean
(상수) (상수이름) = 값
```


###  변수 : 변화가 가능한 수
 -   사용방법 : var (이름) : <Type> : 값
 
 -   사용방법 : 
 ```
 1. var (이름) : <Type> : 값
 2. var (이름) = 값
 ```
 -   ex)
 ``` 
1.  var name : String =  Bean
 (변수) (변수이름) : <타입> =  값
2.  var name2  = Bean
 (변수) (변수이름) = 값
 ```

###  차이점
 1. 상수는 변경이 되지 않아 값 변경을 할 수 없음 
 2. 변수는 변경이 가능 해 값을 변경 할 수 있어 나중에 값을 넣어도 됨
